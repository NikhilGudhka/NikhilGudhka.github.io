arrow_forwardJava Byte Code Parser
Java
Bytecode Parsing
Software Metrics
LCOM4
CBO

Java program to read and parse one or more java .class files provided via command-line parameters and calculate serveral metrics based on the compiled code. The Metrics calculated are Lack of cohesion in methods 4 (LCOM4) and Coupling Between object classes (CBO). The program handles files provided with or without a path component and only parses Java .class files version 51.0 and higher corresponding to Java 7. It has been coded to handle all potential errors, giving the user appropriate error messages. The program produces both terminal and text file output. The terminal output displays a summary of each class's LCOM4 and the overall CBO, Class names fully translated i.e.: public, final, extends etc. , Method names fully translated i.e.: flags, return and parameter types, and Field names fully translated i.e flags, type and names. The text file is a Graphviz/Neato compatible file specifying the graphical representation of each class's LCOM4 diagram and the CBO diagram for all classes.
